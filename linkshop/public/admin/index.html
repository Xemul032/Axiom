Как настроить

  1. Напишите @BotFather в Telegram → /newbot
  2. Создайте два разных бота — один для админов, второй для пользователей
  3. Вставьте токен пользовательского бота в поле выше и сохраните
  4. Пользователь пишет боту /start → нажимает «Зарегистрироваться»
  5. Вводит имя и фамилию — бот сопоставляет с базой и привязывает его chat_id
  6. После регистрации доступна кнопка «Список покупок»
