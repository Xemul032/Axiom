# Монтаж — агрегатор монтажных работ

Система для оформления заявок на монтаж/замер с административной панелью и Telegram-уведомлениями.

## Быстрый старт

```bash
cd montaj
npm install
npm start
```

Сервер запускается на порту **3001**.

- **Админ-панель:** http://localhost:3001/admin  
- **Фрейм (форма):** http://localhost:3001/frame  
- **Логин:** `admin` / **Пароль:** `admin123`

## Tampermonkey

Установи скрипт `montaj-simprint.user.js` в Tampermonkey.  
В начале скрипта укажи реальный адрес сервера:

```js
const SERVER_URL = 'http://YOUR_SERVER_IP:3001';
```

## Структура

```
montaj/
├── server/
│   ├── index.js          — Express-сервер (порт 3001)
│   ├── db.js             — SQLite база (better-sqlite3)
│   ├── telegram.js       — Telegram-уведомления
│   └── routes/
│       ├── requests.js   — API заявок (пользователи)
│       └── admin.js      — Admin API (Basic Auth)
├── public/
│   ├── frame/            — Фрейм с формой заявки
│   └── admin/            — Админ-панель
├── data/                 — БД и загружаемые файлы (создаётся автоматически)
└── montaj-simprint.user.js — Tampermonkey скрипт
```

## Функционал

### Пользователь
- Форма заявки: вид выезда (Монтаж/Замер), дата, тип работ, что монтируем, адрес, контакты, комментарий, файлы
- Автоматическое считывание имени и номера заказа со страницы cplink.simprint.pro
- Редактирование уже отправленной заявки
- Подсказка о доступности выбранной даты

### Администратор
- **Календарь** — обзор заявок по месяцам, клик по дню открывает список
- **Заявки** — таблица с фильтрами, просмотр деталей, смена статуса, удаление
- **Виды работ** — добавление/редактирование/скрытие видов работ
- **Лимиты** — количество монтажей/замеров в день, суммарный лимит, буфер отсрочки
- **Блокировки** — заблокировать/разблокировать дни для новых заявок
- **Telegram** — настройка Bot Token и нескольких Chat ID; тест отправки

## API

| Метод | Путь | Описание |
|-------|------|----------|
| GET | /api/requests/work-types | Список видов работ |
| GET | /api/requests/available-dates?type=montage | Доступные даты |
| POST | /api/requests | Создать заявку |
| GET | /api/requests/:id | Получить заявку |
| PUT | /api/requests/:id | Изменить заявку |
| GET | /api/admin/... | Admin API (Basic Auth) |
