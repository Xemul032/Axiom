## 📖 Описание
Проект направлен на полную модернизацию legacy-скрипта управления менеджерскими процессами. Основная цель — переход от монолитной, сложно поддерживаемой кодовой базы к модульной архитектуре с возможностью безопасного и постепенного внедрения изменений через **Feature Flags**.

## 🎯 Ключевые цели
- ✅ **Декомпозиция монолита:** Разделение единого скрипта на независимые, тестируемые модули (фичи).
- ✅ **Feature Flags:** Внедрение системы управления флагами для контроля доступности функционала без пересборки и деплоя.
- ✅ **Рефакторинг легаси:** Устранение технического долга, улучшение читаемости, стандартизация обработки ошибок и логирования.
- ✅ **Безопасность & Стабильность:** Возможность мгновенного отключения проблемных фич в production, минимизация рисков при обновлениях.

## 🔧 Использование

Каждый модуль представляет собой самодостаточную функцию, которая принимает три параметра:
- `config` — объект конфигурации
- `GM` — объект GM API (Tampermonkey)
- `utils` — общие утилиты

Пример подключения модуля:

```javascript
(function() {
    const config = {/* ваша конфигурация */};
    const GM = window.GM || {};
    const utils = window.utils || {};
    
    // Подключение модуля
    if (config.features.montages?.enabled) {
        loadScript(config.features.montages.url, () => {
            console.log('Модуль montages загружен');
        });
    }
})();
```

## 🚀 Добавление новой фичи

1. Создайте новый файл в папке `features/` (например, `myfeature.js`)
2. Оберните код в IIFE с параметрами `(config, GM, utils)`
3. Добавьте проверку зависимостей в начале
4. Обновите `config.json`, добавив новую фичу
5. Протестируйте и задокументируйте

Шаблон нового модуля:

```javascript
(function(config, GM, utils) {
    'use strict';
    
    // Проверка зависимостей
    if (!GM || !GM.xmlhttpRequest) {
        console.error('[MyFeature] ❌ GM API не передан');
        return;
    }
    console.log('[MyFeature] 🚀 Модуль запущен');
    
    // Ваш код здесь
    
})(
    typeof config !== 'undefined' ? config : {},
    typeof GM !== 'undefined' ? GM : {},
    typeof utils !== 'undefined' ? utils : {}
);
```
